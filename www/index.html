Quick Patient Counseling

About

This app contains the information of drugs and enables users to make notes on specific drugs.

Fast searching for finding quick information of any drugs.

How to use
Updating
Updating is a 2 to 3 step process depending on the difference between local files and the server. Press the update button in 'About' page. The process will guide you through the updation. Press finish in the end to complete updation.
Main Page
Contains the main search bar and two links to 'all drugs' and 'information/help'.
  1. Start the search by typing into the search bar.
  2. Search results will be shown as you are typing.
  3. Tap a single drug to go to its details.
  4. Press 'clear' to clear the search bar.
  5. Press 'back' to go to the main page.
All Drugs
  1. Shows list of all drugs.
  2. Each page shows 50 drugs.
  3. Link to search and back to the main page.
Single Drug Details
  1. Shows details of a single drug.
  2. Tap and hold Keywords to see their description.
  3. If no notes are added to a drug, tap the '+' icon to add a note.
  4. If notes have been previously added, then tap the 'pen' icon to edit notes.
